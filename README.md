# 🏎️ Kart Legends

A fast, fun, and competitive 3D go-kart racing game built with pure HTML, CSS, JavaScript, and Three.js.

No npm. No build tools. Just open and play.

---

## Features

### 🏁 Career Mode
- Start with a basic rental kart.
- Earn coins by racing.
- Unlock faster karts.
- Upgrade engines, tires, handling, and brakes.

### ⚡ Arcade Mode
- Boost pads
- Speed boosts
- Drifting
- Jumps
- Fun arcade gameplay

### 🚗 Normal Mode
A realistic go-kart experience without arcade power-ups.

### 🌍 Multiplayer
- Race friends
- Public matchmaking
- Private rooms
- Cross-platform support (planned)

### 🤖 Offline AI
Practice against AI racers with different personalities:
- Beginner
- Balanced
- Aggressive
- Defensive
- Champion

### 🛠 Customization
- Kart colors
- Helmets
- Suits
- Wheels
- Decals
- Trails

### 🏆 Ranked Seasons
Bronze → Silver → Gold → Diamond → Champion

### 🎁 Rewards
- Daily rewards
- Achievements
- Coins
- Cosmetic unlocks

### 📹 Replay Mode
Watch races again with cinematic cameras.

### 📸 Photo Mode
Pause the race and capture your favorite moments.

### 🏗 Track Creator
Create, test, and share your own tracks.

---

## Controls

### PC

- W / ↑ = Accelerate
- S / ↓ = Brake
- A / ← = Turn Left
- D / → = Turn Right
- Space = Boost (Arcade)

### Mobile

- On-screen steering buttons
- Brake button
- Boost button

## Vision

Kart Legends aims to combine realistic go-kart handling with fun arcade gameplay.

Players can enjoy quick offline races, compete online with friends, climb ranked leaderboards, unlock new karts, and eventually participate in tournaments.
